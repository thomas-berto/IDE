Notepad++

https://notepad-plus-plus.org/downloads/
Notepad++ l’excellent éditeur de texte et très complet, est un éditeur de code source qui prend en charge plusieurs langages Logiciel gratuit sous licence GPL, Notepad++ est un éditeur de code source, simple d’utilisation, avec une interface en français ou bien d’autres langues.
Notepad++ est un éditeur de code qui supporte de nombreux langages de programmation : HTML, ASP, PHP, C, C++, COBOL, CSS, Javascript, XML, etc. Grâce à sa coloration syntaxique, les développeurs auront la tâche facile.
Les 55 langages ci-après sont nativement supportés par Notepad++ (coloration syntaxique, indentation, etc.) depuis la version 6.6.6. Pour les autres langages, l'utilisateur dispose de tous les outils pour paramétrer lui-même un comportement de notepad++ propre à un autre langage (illustration par quelques captures d'écrans ci-après). Les 55 langages dont la syntaxe et la coloration syntaxique sont supportées nativement . Fonctionnalités

  • Coloration Syntaxique et Relief Syntaxique (Syntax Folding)
  • Langage défini par utilisateur
  • PCRE (Perl Compatible Regular Expression) pour la recherche et le remplacement
  • Document Map
  • Auto-complétion
  • Edition multilignes (simultanées) dans des documents colonnés
  • Multi-documents (Les onglets)
  • Multi-vues
  • WYSIWYG (What You See Is What You Get - Ce que vous voyez est ce que vous obtenez - pour l'impression)
  • Zoom in et zoom out
  • Multilingue (dont le français)
  • Signet
  • Enregistrement de Macros et playback
